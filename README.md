# Form Sederhana PHP (OOP)

Project ini merupakan implementasi form sederhana berbasis web menggunakan PHP dengan pendekatan Object-Oriented Programming (OOP). Data yang diinput oleh pengguna akan dikirim menggunakan method POST, kemudian diproses melalui class pada file terpisah, dan hasilnya ditampilkan kembali di halaman utama.

## Fitur
- Form input: firstname, lastname, phone number, dan address
- Pengolahan data menggunakan class, constructor, dan method (OOP)
- Menampilkan hasil input setelah form disubmit
- Tampilan sederhana dan responsif menggunakan CSS

## Struktur File
- index.php → menampilkan form dan hasil output
- process.php → berisi class dan logic pemrosesan data
- style.css → mengatur tampilan antarmuka

## Cara Menjalankan
1. Jalankan Apache melalui XAMPP
2. Simpan folder project di `htdocs`
3. Akses melalui browser: http://localhost/form-sederhana/index.php

## Tujuan
Project ini dibuat untuk memahami dasar pembuatan form, alur pengolahan data, serta penerapan konsep OOP dalam PHP.
